01 — OVERVIEW

Theory met
practice.

In my second year I used current frameworks to build online apps while learning about cloud deployment and scalability. I sharpened my grip on algorithms and data structures to write more efficient code and solve problems more cleanly.

Working on collaborative projects pushed my abilities in version control, software design and team workflow. Alongside the engineering, I doubled down on game development with the Godot Engine and C#, plus C++ for creative audio software — building a portfolio of shipped projects rather than just exercises.

02 — MODULES & PROJECTS

Selected work

CREATIVE GAME ENGINE FEATURED

Max Slinger 3D

Built in the Godot Engine — a physics-based gaming experience that lives or dies on precision and timing. The project meant working with 3D environments, character movement and object interactions, using Godot's physics engine for smooth, dynamic play. It gave me real experience in 3D scene construction, collision handling and camera control, deepening my understanding of game design and user feedback in 3D.

Godot GDScript 3D Physics
WEB

Dynamic Web Applications

The technology behind client-server applications — how websites and web apps actually function, and how data is used to make them behave dynamically. Heavy focus on database systems: data modelling and implementation in SQL alongside NoSQL alternatives.

C++ AUDIO

Extended C++ · DJ Application

Implementing programs with sophisticated techniques — template programming, polymorphism, lambda functions. Examining, extending and applying existing C++ code bases written for creative scenarios, including building a working two-deck DJ application with a custom GUI.

CREATIVE COMPUTING

Whiskers of War

A second Godot game, built around quick decisions and evocative storytelling on the theme of "On and Off." Used Godot's node-based architecture and GDScript for opponent behaviour, player interactions and environmental design — exploring binary states as both a mechanic and a metaphor.

🔌
EMBEDDED

Creative Embedded Programming

A foundation in microcontrollers and how they integrate within a larger computing and creative setting. Covered fundamental electronics, sensor systems and real-time software development using strongly-typed compiled C++.

GAMES

Game Projects · JavaScript

In-engine footage from Whiskers of War alongside complementary JavaScript game experiments — exploring 2D mechanics, interaction loops and the contrast between engine-driven and browser-driven game development.

CREATIVE

Comics & Graphic Novels

The essentials of telling stories through images — simplifying and clarifying ideas, developing recognisable characters, and carrying them through a journey across comic panels. A change of pace alongside the engineering modules, sharpening visual storytelling.
