Second Year Overview

In my second year, I used current frameworks to create online apps while also learning about cloud deployment and scalability. I improved my grasp of algorithms and data structures in order to be more efficient while solving problems and writing code. Working on cooperative projects also helped me improve my collaboration, version control, and program design abilities. I have also enhanced on my game development skills using Godot Engine and C#.

Key Modules & Projects

Dynamic web applications

The technology behind client-server applications is the main topic of this subject. I now understand how websites and web apps function and how data may be utilised to make such apps behave dynamically. The usage of database systems, including an introduction to data modelling and database implementation in SQL and alternatives like No SQL, is a major component of this topic.

Extended C++

Using sophisticated methods to implement programs, such template programming, polymorphism, and lambda functions. Examine, expand, and apply current C++ code bases created for creative scenarios to address issues.

Creative game engine development

I used the Godot Engine to build Max Slinger 3D, with the goal of creating a physics-based gaming experience that relied heavily on precision and time. The project entailed working with 3D environments, character movement, and item interactions, all while utilising Godot's physics engine to generate smooth and dynamic dynamics. This project provided me with great experience in 3D scene construction, collision handling, and camera control, which helped me better grasp game design and user feedback in three dimensions.

Creative embedded programming

provides a foundation for me to get a grasp of microcontrollers and how they integrate within a larger computer and creative setting. It will cover fundamental electronics, sensor systems, and real-time software development using the highly typed compiled language C++.

Creative Computing Project 2

Using the Godot Engine, I created Whiskers of War with the goal of producing an interesting gaming experience centred on quick decisions and evocative storytelling. I experimented with Godot's node-based architecture during development, wrote GDScript scripts, and implemented features including opponent behaviour, player interactions, and environmental design using the theme On and Off. In our digital age, the terms "on" and "off" are more than just binary states; they are fundamental aspects that create technology, art, and human connection. These two states control anything from basic electrical switches to complicated computing algorithms.

Comics and graphics novel

I've mastered the essentials of telling tales via images, including how to simplify and clarify concepts, develop consistent recognised characters, and take those characters on a trip across comic panels. These seminars are geared for students with varying degrees of writing and illustration expertise.
