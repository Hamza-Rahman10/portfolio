Introduction to Programming

Learned the basics of Python, logic building, and algorithmic thinking. Built simple applications like a calculator and a number guessing game.

Foundations of computing 1

Created static websites using HTML and CSS. Explored web accessibility and responsiveness for mobile-first design.

Introduction to mathematics for Scientists

Covered discrete maths, logic gates, and Boolean algebra — foundational for understanding computer architecture. Topics include arithmetic and fractions, linear and quadratic functions and their graphs, exponentials and logarithms, and trigonometry.

Further mathematics for Scientists

Introduced to calculus, which includes differentiation and integration. Differentiation can be used to find the slope of a graph, which tells you how quickly one quantity varies as you change another. Integration allows you to find the area under the curve on a graph, and can be used for quantifying cumulative effects.

Foundations of computing 2

Collaborated in a small team to design and present a timetable planner web app. Developed teamwork, presentation, and feedback-handling skills.

Introduction to systems design

Introduced to the concept of the lifecycle approach to systems development and basic systems analysis and design techniques using a top down, functional decomposition approach.
