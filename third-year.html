FINAL-YEAR PROJECT FEATURED

Final-Year Project & Dissertation

This project is a third-person action game prototype developed in Unity and set within a dystopian futuristic wasteland. The game follows a young survivor named Simon as he navigates a collapsing world filled with hostile soldiers, destruction, and scarce resources. The project focuses on creating engaging melee combat through the use of state machine-driven player mechanics, enemy AI behaviour, dodging, and targeting systems. A central gameplay feature is the dynamic balance between speed and strength, allowing players to alter their combat style strategically by sacrificing one attribute to enhance the other. Through its gameplay systems, environmental storytelling, and narrative cutscenes, the project aims to create an immersive and tense survival experience.

AI / ML

Data & Machine Learning

Embodied Digital Organism is a real-time interactive installation where your body controls a living particle creature through your webcam. Using Google's MediaPipe to detect body landmarks, the system translates gestures like raising arms, lowering them, bringing hands together, moving side to side into the behaviour of an amoeba-like creature built from hundreds of particles, complete with a glowing core, flowing tentacles that reach toward your hands, and rainbow colours that shift with movement.

AI

Game AI Programming

This project focuses on designing and evaluating an AI-controlled agent within a game environment using Behaviour Trees and pathfinding techniques. The aim is to create an intelligent Pac-Man AI capable of balancing multiple objectives, such as collecting pellets, avoiding ghosts, and adapting to changing game states. Through iterative development and automated testing, the project explores how different AI behaviours impact overall performance, survival, and score efficiency. The final solution is supported by technical analysis, behaviour diagrams, testing results, source control evidence, and critical reflection on the strengths and limitations of the implemented AI system.

SYSTEMS

Interaction Design

This project explores the design of an AI-supported educational platform that helps university students improve self-regulated learning and metacognitive awareness through a structured, visual workflow rather than traditional text-only AI chat. The system combines AI-generated summaries, mindmap-based interaction, adaptive planning, and lightweight reflection prompts to help students better understand concepts, organise study tasks, and monitor their own learning progress. Through iterative prototyping, usability testing, and behavioural analysis, the project investigates how visual knowledge structures and mixed-initiative AI interactions can reduce cognitive overload and create a more effective and engaging university learning experience.

SIMULATION

Virtual Reality & 3D Environment

This project is an interactive VR anatomy lab developed in Unity that allows users to explore the human body within an immersive virtual environment. The experience was designed to make learning anatomy more engaging and accessible through 3D spatial interaction, enabling users to examine skeletal structures and anatomical elements in greater detail than traditional learning methods. The project incorporates VR interaction systems, world-space UI, anatomical labeling, and optimised 3D assets to create a cohesive educational experience that combines both technical implementation and visual design.
